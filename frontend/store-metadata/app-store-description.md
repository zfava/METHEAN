# METHEAN — App Store Description

## Short Description (30 chars)
Homeschool learning, governed.

## Full Description

METHEAN is a learning operating system for homeschool families.

You set the rules. AI follows them.

- Build structured learning maps with prerequisite enforcement
- Track real mastery with FSRS spaced repetition (not just completion)
- AI tutoring, planning, and evaluation governed by your family's values
- 51-state compliance tracking with document generation
- Every AI decision logged, inspectable, and overridable
- Evaluator calibration that proves its own accuracy over time

METHEAN doesn't replace you as the teacher. It gives you the instruments to govern your children's education with precision and confidence.

Built for families. Governed by parents. Powered by AI.
